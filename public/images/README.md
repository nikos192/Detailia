# Portfolio Images

Place your car detailing portfolio images in this folder. The website expects the following image files:

## Required Images

1. **car-1.jpg** - Premium Exterior Detailing shot (recommended: 800x600px or larger)
2. **car-2.jpg** - Mercedes AMG Detail Work (recommended: 800x600px or larger)
3. **car-3.jpg** - Lamborghini Showroom Finish (recommended: 800x600px or larger)
4. **car-4.jpg** - High-End Paint Protection example (recommended: 800x600px or larger)

## Instructions

1. Download or obtain your high-quality car detailing photos
2. Rename them according to the names above (car-1.jpg, car-2.jpg, etc.)
3. Save them in this `/public/images/` folder
4. The website will automatically display them in the gallery section

## Image Specifications

- **Format**: JPG or PNG
- **Resolution**: Minimum 800x600px (larger is better for quality)
- **Aspect Ratio**: Landscape (wider than tall) preferred
- **File Size**: Keep under 5MB each for optimal performance

## Tips

- Use high-quality photos that showcase your best work
- Ensure good lighting and clear visibility of the vehicle details
- Before/after comparisons work great!
- Include vehicles with the Detailia branding when possible

## Updating Images

The website references images at these paths:
- `/images/car-1.jpg`
- `/images/car-2.jpg`
- `/images/car-3.jpg`
- `/images/car-4.jpg`

Simply replace the files in this folder to update the gallery without touching any code.
